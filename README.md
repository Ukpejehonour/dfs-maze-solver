# 🧩 DFS Maze Solver - Python Turtle

A visual maze-solving program that uses Depth-First Search 
(DFS) algorithm to autonomously navigate and solve mazes,
built entirely with Python Turtle.

## 🎯 What It Does
- Solves mazes autonomously using DFS pathfinding
- Visually draws each step of the solving process
- Uses dictionary-based memory to track visited paths
- Prevents revisiting already explored routes

## 🛠️ Technologies Used
- Python
- Python Turtle library
- Depth-First Search (DFS) algorithm
- Dictionary data structures for path memory

## 📁 Files In This Repository
- `main.py` — Main DFS algorithm and visual logic
- Screenshots — Visual proof of maze being solved
